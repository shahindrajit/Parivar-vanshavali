# Parivar-vanshavali
# 🪔 Vanshavali – Hindu Family Tree (Bansawali) Builder

**Vanshavali** is a free, privacy‑first, client‑side Hindu Family Tree (Bansawali / Vanshavali) builder built using pure **HTML, CSS, and JavaScript**.  
It allows users to create, visualize, manage, export, and share detailed family lineage charts directly in the browser—without any server or backend.

🔒 **All data stays on the user’s device** using browser local storage.

## ✨ Key Features

- 🌳 **Interactive Family Tree Builder**
- 👥 Add ancestors, descendants, spouses & generations
- 🎯 Focus Lineage Mode (isolate a branch)
- 🔍 Generation highlighting
- 🌐 **Bilingual Support** – English & हिन्दी
- 🌗 Light & Dark themes
- 🔗 Kinship (Rishta) Calculator
- 📊 Family Insights Dashboard
- 🗓️ Upcoming dates (Birthdays / Punya Tithi)
- 💾 Multi‑tree support (store multiple family trees)
- 🔐 Privacy‑first (no cloud, no tracking)
- 📤 Export to:
  - JPG (Image)
  - PDF (Horizontal)
  - CSV
  - JSON (Backup)
- 🔗 Share family tree via encoded link
